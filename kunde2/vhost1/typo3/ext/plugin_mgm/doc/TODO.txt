**************
STATUS
**************
This module was originally intended to be a framework for all plugins in a branch of the page tree. However it has never taken off and the original significance of it may be removed soon.
One way to develop the module would be to make it a framework for plugins (like the Template module is for the "Function menu modules" there) which has an API so extensions with plugins could add management functions into the plugins module. I think this should be very attractive alternative to putting a similar module into "Info" or "Functions" where such a thing does not belong if it is directly related to a plugin.
